PyWrithe Documentation
==================================

.. toctree::
   :hidden:
   :maxdepth: 1

PyWrithe is a all python package to compute writhe of closed curves.
Supports both numpy and jax.numpy for jit.

How to install
''''''''''''''

.. prompt:: bash $ auto

  $ python3 -m pip install pywrithe

and you'll be good to go.
